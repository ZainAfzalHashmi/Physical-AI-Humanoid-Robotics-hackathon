# Glossary

This appendix contains definitions of technical terms used throughout the book.

## Terms to be defined:
- ROS 2: Robot Operating System 2 - A flexible framework for writing robot software
- Node: An executable that uses ROS 2 to communicate with other nodes
- Topic: A named bus over which nodes exchange messages
- Service: A synchronous request/response communication pattern
- Publisher: A node that sends messages to a topic
- Subscriber: A node that receives messages from a topic
- URDF: Unified Robot Description Format
- Xacro: XML Macros for URDF
